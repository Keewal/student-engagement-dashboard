# Student Engagement Dashboard Prototype

## Overview
This prototype is built with Python and Streamlit to demonstrate:
- KPI Calculation (Engagement Rate, Drop-off Rate)
- AI-based insight for at-risk students
- Interactive buttons for actionable reminders

## How to Run
1. Install Streamlit if not already installed:
   pip install streamlit

2. Run the app:
   streamlit run app.py

## Files
- student_activity.csv: Sample input data
- app.py: Streamlit dashboard code
- AI_prompt_documentation.txt: AI usage details
